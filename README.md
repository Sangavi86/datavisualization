# 🎬 Netflix Analytics & AI Recommendation Platform

## 📌 Project Overview

Netflix has a massive collection of movies and TV shows. Understanding content trends, audience preferences, ratings, genres, and business insights can help improve decision-making and recommendation systems.

This project is an end-to-end **Netflix Analytics & AI Recommendation Platform** built using **Python, Streamlit, Machine Learning, and Data Analytics** techniques.

The platform allows users to upload Netflix data, explore interactive dashboards, generate business insights, analyze content trends, receive AI-based recommendations, and create professional reports.

## ▶️ How to Run the Project

### Step 1: Clone Repository

```bash
git clone https://github.com/Sangavi86/datavisualization.git
```

### Step 2: Move into Project Folder

```bash
cd datavisualization
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run Application

```bash
streamlit run app.py
```

### Step 5: Open Browser

```text
http://localhost:8501
```
